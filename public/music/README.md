# Default Theme Music

Place your default theme songs in this folder with the following names:

- `birthday.mp3` - Default song for Birthday theme
- `christmas.mp3` - Default song for Christmas theme
- `neutral.mp3` - Default song for Neutral theme (optional)

These songs will play automatically when the corresponding theme is selected in View Mode.

## Supported Formats

- MP3 (recommended)
- WAV
- OGG

## Recommended Settings

- Bitrate: 128-320 kbps
- Sample Rate: 44.1 kHz
- Keep file sizes reasonable (< 5MB) for faster loading
